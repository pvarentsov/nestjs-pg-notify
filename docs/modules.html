PgNotifyConnectionStrategy

PgNotifyConnectionStrategy: { retryInterval?: number | ((attempt: number) => number); retryLimit?: number; retryTimeout?: number }
  • Defined in src/pg-notify/pg-notify.type.ts:10

Type declaration

  • Optional retryInterval?: number | ((attempt: number) => number)

    How much time to wait between reconnection attempts (if failed). Can also be a callback returning a delay in milliseconds. Defaults to 500 ms.

  • Optional retryLimit?: number

    How many attempts to reconnect after connection loss. Defaults to no limit, but a default retryTimeout is set.

  • Optional retryTimeout?: number

    Timeout in ms after which to stop retrying and just fail. Defaults to 3000 ms.
